O que é um arquivo html?
    -É um arquivo de texto com marcações html que dão significado ao conteudo.

-Titulo vs paragrafo
    <h1> Isso é um título </h1>
    <p> Isso é um paragrafo </p>
    <p> Isso é um parágrafio com <b>negrito</b></p>
Obs:Primeira tag aberta é a ultima fechada

-Para criar arquivo .html se usa editor de texto
-Para ler utilizamos navegadores

-.html = Página | diretorio = site
